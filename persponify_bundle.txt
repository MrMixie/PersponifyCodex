Persponify Bundle (REFERENCE ONLY)

This repo never stores live context bundles or script sources.
Bundles are generated locally from Studio exports and must stay local.

Purpose:
- Document the bundle shape for tools.
- Provide a safe, redacted template.

Fields (high level):
- meta: buildId, placeId, gameId, pluginVersion, caps, fingerprint, and truncation flags
  (truncatedBySize, totalCapHit, omittedByDiff/Size/Total, attributesIncluded, tagsIncluded).
- tree: instance paths + class names.
- scripts: path metadata; source may be omitted or truncated.

Template (redacted):
=== META ===
buildId: BUILD-YYYY-MM-DD-...
pluginVersion: PersponifyCodex/x.y.z
placeId: 123
gameId: 456
maxScriptChars: 120000
maxTotalChars: 400000
fingerprint: fnv1a32:...
truncatedBySize: false
totalCapHit: false
omittedByDiff: 0
omittedBySize: 0
omittedByTotal: 0
attributesIncluded: true
tagsIncluded: true

=== TREE (PATHS) ===
- game/ReplicatedStorage [ReplicatedStorage]
- game/ServerScriptService [ServerScriptService]

=== SCRIPTS (METADATA ONLY) ===
- path: game/ReplicatedStorage/SomeModule
  className: ModuleScript
  bytes: 1234
  sha1: sha1:...
  source: (omitted)
  sourceTruncated: false
  attributes: (optional)
  tags: (optional)
